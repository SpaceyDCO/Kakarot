# Kakarot

A totally configurable Minecraft plugin for servers on 1.7.10.
This plugin connects to the mod CustomNPCs+ (By Kamkeel) and is intended to be used on servers with the mod Dragon Block C.

## ⚙️ Tech Stack
*   **Data Management:** Structured YAML/JSON configuration handling, as well as an SQLite database file
*   **Optimized:** Built with server TPS and performance in mind

## 🚀 Key Mechanics
*   Tower Defense system
*   Quests System (with support for localized messages)
*   "Phasing NPCs system". A unique WoW-like system that prohibits some players from seeing certain NPCs. (i.e. Player can't see "Goku" if they don't have the permission "quest.goku.npc")
